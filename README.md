# nodejs-express-tasks-api-topic4

## Project Overview
This project is a Tasks REST API developed using Node.js and Express.js.  
The API performs CRUD operations for task management.

## Features
- Create a new task
- View all tasks
- Update existing tasks
- Delete tasks
- JSON request and response handling
- Express routing and middleware

## Technologies Used
- Node.js
- Express.js
- npm
- Postman

## Steps Followed
1. Initialized the Node.js project using `npm init`
2. Installed Express.js using npm
3. Created the server file
4. Configured Express middleware
5. Implemented REST API routes
6. Tested endpoints using Postman
7. Uploaded the project to GitHub

## API Endpoints
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

## Challenges Faced
- Handling JSON request parsing
- Fixing route path issues
- Resolving server port conflicts

## Outcome
Successfully built and tested a working Express.js Tasks REST API application.
